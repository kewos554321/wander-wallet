# Tech Stack

## Frontend
- Next.js 16 (App Router)
- React 19.2
- TypeScript 5
- Tailwind CSS v4
- Radix UI

## Backend
- Next.js API Routes
- Prisma 6.18 ORM
- PostgreSQL (Neon)

## Auth
- LINE LIFF SDK
- JWT strategy

## AI
- LangChain + DeepSeek V3 (voice parsing)
- Gemini 2.0 Flash (receipt OCR)

## Other
- Storage: Cloudflare R2
- Maps: Leaflet + React Leaflet
- Charts: Recharts
- PDF: jsPDF + jspdf-autotable

## Path Alias
- `@/*` maps to project root
